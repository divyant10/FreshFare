# 🛒 FreshFare

FreshFare is a live grocery e-commerce website built to provide a seamless online shopping experience for daily essentials.

🌐 **Live Website:**  
👉 https://freshfare.infinityfreeapp.com

---

## 🎯 Problem

Local grocery shopping often lacks:

- Convenience  
- Online ordering system  
- Digital payment options  
- User account tracking  

---

## 💡 Solution

FreshFare enables users to:

- 🛍️ Browse grocery products  
- 🛒 Add items to cart  
- 🔐 Secure login & signup  
- 💳 Choose payment methods (UPI / COD / Online Banking)  
- 📦 Complete checkout process
